## 二、按需求条目对照

### 🔴 P0 — 网关核心兜底（需求 #1/#3）

- **thinking 三态补全**（`applyGatewayCompat`）：passthrough / inject / strip。
  inject 时注入 `{type:'enabled', budget_tokens:N}`、把 `max_tokens` 抬到
  `budget+1024`、按 Anthropic 规范移除 `temperature/top_p/top_k`；支持按
  route 配置独立预算（`routeBudgets`）。
- **SSE 流式健壮性**（`forwardToUpstream`）：25s 心跳防断连、流结束补
  `event:done / [DONE]`、上游中断写合规 `event:error`、客户端断开销毁上游并
  清理定时器、流式 10 分钟 / 非流式 2 分钟超时。
- **错误归一化**：上游 ≥400 一律转成 Anthropic `{type:'error',error:{type,message}}`；
  `classifyUpstreamError`（rate_limit/model_not_found/thinking_unsupported/…）+
  `classifyConnectionError`（ECONNREFUSED/ENOTFOUND/超时/SSL → 中文提示）。
- **脱敏日志**（`gatewayLog`，默认关）：仅记时间戳+路径+模型改写前后，
  绝不记 body/key；写入 `gateway-requests.log`，超 512KB 滚动留最近 1000 行。

### 🔴 P0 — 跨平台 macOS（需求 #5）

- 路径：`HOME=os.homedir()`，`PATHS` 按 `IS_WIN/IS_MAC` 分支
  （macOS 用 `~/Library/Application Support/Claude-3p`、`ccx-desktop`）。
- 进程：`isProcessRunning`(tasklist/pgrep)、`isPortListening`(netstat/lsof)、
  `killProcessByName`(taskkill/pkill)、`launchApp`(spawn exe / `open -a` .app)、
  `sleepSync`(timeout/sleep)。
- 自启动：Windows 启动文件夹 VBS / macOS `~/Library/LaunchAgents` plist +
  `launchctl`；`/api/autostart` 返回 `platform`。
- 脚本：新增 `start.sh` / `stop.sh`。

### 🟡 P1 — 健壮性

- **Codex CLI TOML**（`writeCodexCli`）：改用 `@iarna/toml`
  parse → 改对象 → stringify，正确处理嵌套表/布尔/转义；解析失败拒绝写入。
- **Codex Desktop modelMapping**（`saveMappingCXD`）：先全量校验
  （必须是 string→string 的 JSON 对象）再写入，任一非法整体取消。

### 🟢 P2 — 收尾

- **导出脱敏**：`exportCDConfig(id, stripKey)` + `?stripKey=1`，前端弹窗
  让用户选「含密钥 / 不含密钥」，去密钥文件名加 `-nokey`。
- **路由空后端校验**：`saveGateway` 保存前提示空后端模型。
- **测试接入 CI**：`npm test` 跑两个测试文件，CI workflow 自动执行。
